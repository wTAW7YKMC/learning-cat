# 学习喵 - 待办计时 + 动漫卡片收集工具

> 面向学生群体的学习辅助工具，结合待办计时与动漫卡片收集，提升学习效率与趣味性

## 🌟 项目特色

- **智能任务管理**：多周期、多维度任务配置，支持优先级和难度设置
- **双向计时系统**：正向计时记录学习时长 + 倒计时提醒截止时间
- **番茄工作法**：25分钟专注 + 5分钟休息，提升学习专注度
- **动漫卡片收集**：主题分类 + 稀有度分级，通过游戏化激励学习
- **数据可视化**：学习时长趋势、学科分布、完成率等详细统计
- **多端适配**：响应式设计，支持Web、移动端、小程序

## 🚀 技术栈

### 前端
- **React 18** + TypeScript
- **Tailwind CSS** + shadcn/ui 组件库
- **Zustand** 状态管理
- **React Router** 路由管理
- **Vite** 构建工具

### 功能模块
- 用户认证系统
- 任务管理（创建/编辑/状态追踪）
- 计时器（正向/倒计时/番茄钟）
- 卡片收集系统
- 数据统计与分析

## 🛠️ 安装与运行

### 环境要求
- Node.js 16+
- npm 或 yarn

### 快速开始

1. **克隆项目**
```bash
git clone <repository-url>
cd learning-cat
```

2. **安装依赖**
```bash
npm install
# 或
yarn install
```

3. **启动开发服务器**
```bash
npm run dev
# 或
yarn dev
```

4. **构建生产版本**
```bash
npm run build
# 或
yarn build
```

## 📱 功能展示

### 核心功能

1. **任务管理**
   - 创建/编辑学习任务
   - 设置难度和优先级
   - 任务状态追踪（待执行/进行中/已完成）

2. **计时系统**
   - 正向计时记录学习时长
   - 倒计时提醒功能
   - 番茄钟工作模式
   - 暂停/继续/重置操作

3. **卡片收集**
   - 4种稀有度：普通/稀有/史诗/传说
   - 按学科和主题分类
   - 解锁条件和收集进度

4. **数据统计**
   - 学习时长趋势图表
   - 学科时间分布
   - 任务完成率分析
   - 个性化学习建议

### 界面特色

- **清新动漫风格**：柔和的蓝色和橙色主题
- **响应式设计**：适配桌面和移动设备
- **动画效果**：卡片翻转、进度条动画等
- **用户友好**：直观的操作流程和反馈
